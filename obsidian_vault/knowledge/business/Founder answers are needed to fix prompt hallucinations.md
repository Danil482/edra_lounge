# Founder answers are needed to fix prompt hallucinations

A 6-question questionnaire was sent to the Defy founders on 2026-04-30. Their answers will unblock Phase 5.1 (the `_defy_brand.txt` fact sheet that [[Empty system prompts cause LLM to hallucinate brand facts|stops the LLM from hallucinating]]).

## The six questions

1. **Anonymized case examples** — 2-3 short anonymized client stories for booth use
2. **Permission to cite founder credentials** — public OK to mention Ian's SHARE/Samy background and Alek's Singapore AI prof?
3. **Out-of-scope boundaries** — 3-5 explicit "we don't do X" statements
4. **Engagement format** — demo → trial → paid pilot? What is the literal next step?
5. **Booth ICP & lead product** — which audience and which of Monitor/Automate/Report to lead with?
6. **Conferences / shared-context anchors** — which events does Defy attend?

## What is blocked

- Phase 5.1 — fact sheet
- Phase 5.2 — prompt rewrite (depends on facts)
- Phase 5.3 — refusal behaviour (depends on knowing what is out-of-scope)
- Phase 5.5 — template alignment (depends on fact sheet)

## What is NOT blocked

- Phase 5.4 — scenario test harness (can lock in baseline on current prompts)
- UI polish tasks
- Tech debt
- Dataset quality verification
